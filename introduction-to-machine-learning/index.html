Introduction to Machine Learning

03 Apr, 2023

Since the emergence of AlphaGo, AlphaFold2, and ChatGPT etc., artificial intelligence technologies (including machine learning) have revolutionalized today’s paradigm of scientific research. However, little is known about the principles of the tech itself by people outside of CS areas. This blog is to briefly introduce these concepts.

Before diving into the fancy/mysterious words, we must first understand what learning is in human beings.

What is learning in humans?

Gain of knowledge from experience.

experience ➡️ learning ➡️ knowledge

If a baby wants to learn the concept of color, he/she needs to “observe” many colorful things.

  • Apple is red;
  • Banana is yellow;
  • Mango is yellow;
  • Blood is red.

The baby then learns how to differentiate red and yellow. Then he/she is given a test.

What’s the color of a rose?

This knowledge of color is then “applied” in other situations.

This is an example of learning from experiences.

What is machine learning?

A type of “learning” without being explicitly programmed.

data ➡️ machine learning ➡️ computer models

If a machine is set to develop a model predicting whether or not a patient has cancer, the machine also requires to “observe” many data.

  • 78-year-old man with 30 years history of smoking has cancer.
  • 67-year-old woman with weight dropping 10kg within a month has cancer.
  • 30-year-old man coughing frequently in the past 2 weeks didn’t have cancer.

A machine learning algorithm learns that one with old age, rapid weight dropping, and long history of smoking may have cancer. Then the algorithm also needs to be tested.

What about a 50-year-old woman started coughing frequently in the past month with TB history?

This knowledge of predicting cancer is then “applied” to test whether it is successfully “learned”.

This is an example of learning from data.

In this process, we don’t explicitly give specific instructions on how to identify cancer patients.

Machine learning algorithms vs Rule-based algorithms

  • ML algorithms: No specific instruction is given.
  • Rule-based algorithms: if, then, else …

Three types of ML

Supervised learning

Requires labeled datasets for training

  • e.g., Differentiate between apple and pear

Unsupervised learning

No labels, discovering clusters

  • e.g., Identify subsets from data

Reinforcement learning

Trained from feedbacks to perform a task

  • e.g., AlphaGo trained to play chess

Artificial Intelligence vs Machine Learning vs Deep Learning

  • AI – machine with human intelligence features (i.e., do things only human can do)
  • ML – learn from data
  • Deep Learning – a type of ML algorithms mimicing neural networks
